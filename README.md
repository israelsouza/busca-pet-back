## Busca pet - Projeto de fins acadêmicos

Busca pet é uma plataforma online dedicada a ajudar na localização de animais de estimação perdidos. Nosso objetivo é conectar donos de pets com a comunidade para aumentar as chances de reencontro.

## Tecnologias e Técnicas Utilizadas

### Stack Principal

| Tecnologia     | Descrição                     |
| -------------- | ----------------------------- |
| **Node.js**    | Runtime JavaScript            |
| **TypeScript** | Superset tipado do JavaScript |
| **Express**    | Framework web minimalista     |
| **PostgreSQL** | Banco de dados relacional     |
| **Docker**     | Containerização de aplicação  |

### Bibliotecas

| Biblioteca    | Finalidade                       |
| ------------- | -------------------------------- |
| **CORS**      | Controle de acesso entre origens |
| **Joi**       | Validação de schemas             |
| **JWT**       | Autenticação stateless           |
| **Sequelize** | ORM (Object-Relational Mapping)  |
| **Jest**      | Framework de testes              |
| **ESLint**    | Linting e qualidade de código    |
