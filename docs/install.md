## 📦 Instalação

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/busca-pet-back.git

# Acesse o diretório do projeto
cd busca-pet-back

# Instale as dependências
npm install

# Inicie o servidor de desenvolvimento
npm run dev

# Acesse a aplicação em http://localhost:3000
```
